# Fog/Clouds

Бро мне максимально лень писать на английском, буду на русичанксом

## Get Started

- Добавь ренедр фичу `Volumetric Fog Feature`
- Создай Global Volume и добавь `Custom/Volumetric Fog` чтобы изменять настройки

## То что нужно знать

- Ты можешь сохранять настройки облаков в preset-ах

## Открыте проблемы

- Я не пофиксил багу с MipMap-ами. В шейдере `VolumetricFog.shader` есть 2 варнинга о том, что из-за циклов Sample 3д
  текстуры может быть неверным. Чтобы пофиксить надо самому считать производные чтобы вычислить правильный Mip уровень,
  но это на потом
